# Arquitectura SGI Auditoria

El sistema separa la gestion de Auditoria de la fuente de datos del ERP mediante adaptadores.

## Capas

- Frontend React
- Backend Express
- Base operativa SGI
- Adaptador ERP
- Motor preventivo de reglas

## Empresa activa

Todas las consultas operativas admiten filtro por empresa. Este criterio se aplica a investigaciones, monitor preventivo, hallazgos, auditorias y exportaciones.
